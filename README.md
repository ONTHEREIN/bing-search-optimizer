# 必应搜索优化器 v7.0.5


一个强大的用户脚本，为必应搜索提供现代化、美观的界面优化，提升搜索体验和视觉舒适度。

## ✨ 主要特性

### 🎨 现代化界面设计
- **双列网格布局** - 搜索结果以双列展示，充分利用屏幕空间
- **圆角设计** - 柔和的圆角边框，现代感十足
- **浅蓝主题** - 清爽的浅蓝色调，保护视力
- **平滑动画** - 优雅的过渡动画效果

### 🌙 智能主题系统
- **深色/浅色主题切换** - 一键切换，护眼舒适
- **与必应原生同步** - 自动检测并同步必应的主题设置
- **主题记忆功能** - 记住您的主题偏好

### 🖼️ 个性化背景
- **自定义背景图** - 支持设置任意图片作为背景
- **内置预设** - 提供多张精美壁纸预设
- **毛玻璃效果** - 背景模糊效果，内容清晰可读

### ⚙️ 便捷操作功能
- **拖拽设置按钮** - 可自由拖拽的悬浮设置按钮
- **回到顶部** - 一键回到页面顶部
- **响应式设计** - 完美适配各种屏幕尺寸

### 🧹 智能内容优化
- **干扰元素屏蔽** - 自动隐藏广告和无关内容
- **AI内容优化** - 智能处理AI聊天和Copilot结果
- **重要功能保护** - 保留搜索框、分类等核心功能

## 🚀 安装方法

### 前提条件
- 安装用户脚本管理器（推荐 [Tampermonkey](https://www.tampermonkey.net/)）

### 安装步骤
1. 点击右侧的 `Releases` 页面下载最新版本
2. 在Tampermonkey中选择"添加新脚本"
3. 将下载的脚本代码粘贴到编辑器中
4. 保存并启用脚本

### 自动安装
[](https://github.com/ONTHEREIN/bing-optimizer/raw/main/bing-optimizer.user.js)

## 🎯 使用方法

### 基本使用
安装后访问 [必应搜索](https://www.bing.com)，脚本会自动生效：
- 搜索结果自动转为双列布局
- 界面应用现代化样式
- 右下角出现设置按钮

### 设置面板
点击右下角的设置按钮打开控制面板：
- **背景图设置** - 输入图片URL或使用预设
- **主题切换** - 在深色/浅色模式间切换
- **实时预览** - 设置立即生效

### 拖拽功能
- 长按设置按钮可自由拖拽到任意位置
- 位置自动保存，下次访问时保持
